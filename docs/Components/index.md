# ProMax 组件

基于 antd 生态，扩展封装的加强组件！功能更丰富，开发效率提升，性能可靠！

## 特性

- 🚀 基于 **@cgf-tools/store** 的局部状态封装，简单小巧高性能；
- 🚥 基于 Hooks 设计理念，优先提供 Hooks Api, 组件扩展性、可用性、灵活性更强；
- 💡 多个相同组件状态互不冲突，解决大型组件封装难题。
- 💎 使用简单、灵活、易于测试
